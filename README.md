# Juego del Vaquero

### Prerequisites
```
UnrealEngine 4.16.x
```

### Ampliaciones
 * Nuevo enemigo animado.
 * Disparos finitos.
 * El jugador podrá lanzar granadas.
 * Zona que ralentiza a un enemigo.
